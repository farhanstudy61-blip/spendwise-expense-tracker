# AI Agents Used

During the development of this project, AI tools were used as a helper to improve productivity and solve development issues.

The main role of AI was to assist with:
- understanding how to structure a full‑stack project
- debugging errors in the React frontend and Flask backend
- suggesting improvements for UI layout and styling
- explaining Git commands for uploading the project to GitHub

AI was treated as a development assistant rather than a replacement for coding. I reviewed the suggestions, adjusted the code when needed, and ensured everything worked correctly in the application.

The goal was to use AI as a learning tool to better understand the technologies involved in this project.
