# Prompting Rules

AI was used to assist during development by asking clear and focused questions.

Some general guidelines followed while prompting AI were:

1. Clearly describe the problem or goal.
   Example: asking how to connect a React frontend with a Flask backend API.

2. Provide relevant context.
   For example, including code snippets when debugging layout or API issues.

3. Ask for explanations rather than only solutions.
   This helped in understanding how the solution works.

4. Review AI suggestions before applying them.
   All generated suggestions were checked and tested before being added to the project.

5. Use AI as a learning tool.
   The purpose was not just to get answers, but also to understand better development practices.

Using these guidelines helped make the AI assistance more effective during development.
